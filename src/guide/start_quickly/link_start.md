## 啊哦，遇到问题了！

遇到问题不要怕，以下是常见问题的解决方案

### Q:进入游戏后卡在城镇页面
A：FAA为识图自动化软件，它在城镇页面时需要能够看到世界地图。请确保你的游戏窗口没有被最小化，浏览器窗口需被拉伸到游戏窗口完全显示为止

### Q:到游戏关卡选择页面后就不动了，它不会进入岛屿
A:FAA仅能识别正常状态下的岛屿图标，请确保岛屿图标被点击过，不是橙色的；极少数情况，新账号的活动图标会阻挡住岛屿图标，此时请给开发组上报错误信息

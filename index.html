    超全功能模块 完整自动化服务

    FAA注重稳定流畅的日常清理, 完成配置后仅需一键即可自动清空一整周的各种事项! 解放您的双手!

    刷新登录

    轻松刷新, 自动登录, 刷新游戏缓存，复位稳定挂机

    自动签到

    一键完成 每日签到 / VIP签到 / 美食活动 / 法老抽奖 / 塔罗抽奖 / 会长发任务 / 营地领钥匙 / 月卡领取

    自定义任务队列

    以任意的顺序, 自由组合 战斗 / 物品清理 / 消耗品使用 / 刷新 / 领取奖励 等功能, 个性自定义.

    常规刷本

    在单一副本完成每天的45场满掉落战斗

    物品清理

    删除各种垃圾物品材料, 清理您的道具背包, 支持自定义

    消耗品使用

    使用各种消耗品, 免去手点烦恼, 支持自定义

    公会任务

    情侣任务

    识别 + 完成 + 领取公会任务

    100%准确度超强OCR文字识别 + 语义解析 + 自动方案生成 + 自动化全流程 = 一键肝完美食大赛

    萌宠神殿

    蚊子腿也完全不放过

    无限跨服

    无限一分钟退出循环刷跨服副本, 威望经验滚滚来

    超强战斗执行器

    最简单的配置, 强大的功能, 极强的方案通用性, 重新挖掘游戏的趣味性

    • 自动化进退

      支持所有地图的全自动进入与退出

    • 自动承载卡

      为90%的地图覆盖全自动的承载卡预设, 支持冷门承载, 您完全不用在战斗方案中配置它们, 仅需将卡片塞入卡组任意位置即可

    • 自动冰沙

      智能且自动的使用极寒冰沙卡片加速布阵, 您完全不用在战斗方案中配置它们, 仅需将卡片塞入卡组任意位置即可

    • 自动复制

      可根据配置的卡片被复制优先级, 自动使用您携带的多种复制类卡片, 加速布阵

    • 状态识别

      基于图像识别的卡片状态识别放卡, 无需对您的游戏练度做任何预设, 即可体验丝滑放卡

    • 轻松分享

      一方案一文件明文储存, 轻松传递您的巧思

    • 高级战斗(beta)

      基于深度学习的敌对目标识别, 以及线性规划的最优放卡策略, 对场地危害和敌方高危目标进行定点清除! 支持GPU加速
